重要なポイント

  • B1：60分、3タスク（非公式メール、フォーラム投稿、正式メール）。B2：75分、2タスク。
  • 試験官は4つの基準で評価します：タスクの達成度、一貫性、語彙、文法。
  • 1つのLeitpunkt（プロンプトポイント）を欠くと、タスクの達成度で大きな得点を失います。
  • 接続詞（trotzdem、außerdem、deshalb）を使用することで、一貫性の得点が向上します。
  • レジスターが重要です：非公式タスクでは「du」を、正式なタスクでは「Sie」を使用してください — 混在すると得点を失います。

レベル別ライティングタスク

Schreibenセクションはレベルによって大きく異なります。以下が完全な概要です：

レベル 時間 タスク 書く内容
B160分3非公式メール/メッセージ、フォーラム投稿、正式メール
B275分2フォーラム投稿（約150語）、正式メッセージ（約100語）
C180分2エッセイ（200語以上）、正式な要約（80〜100語）

すべてのレベルで、ライティングは人間の試験官によって評価されます — 機械ではありません。これは、彼らが真のコミュニケーション能力、多様な言語、明確な構造を評価することを意味します。また、単に暗記したテンプレートでシステムを操作することはできません。

B1ライティング：全3タスクの説明

タスク1：非公式メールまたはメッセージ

内容：友人にメールまたはWhatsAppスタイルのメッセージを書き、彼らのメッセージに応答します。プロンプトで与えられた3つの特定のLeitpunkte（内容ポイント）に言及する必要があります。

試験官が期待すること：

  • 全体を通して非公式なレジスター（du、dein、"Liebe/r..."のような非公式な挨拶）
  • 3つのLeitpunkteが明確に言及されていること
  • 自然なトーン — 実際に友人に書いているかのように
  • 適切なオープニングとクロージング（"Sehr geehrte Damen und Herren"ではない！）
  • 最低60〜80語

例のプロンプト：あなたの友人が新しい都市に引っ越すことについて書いてきました。返信して、（1）ニュースに反応し、（2）新しいアパートについて尋ね、（3）彼らを訪れることを提案してください。

タスク2：フォーラム投稿

内容：オンラインフォーラムのディスカッションプロンプトに応答します。意見を表明し、経験を共有し、トピックに関与します。

試験官が期待すること：

  • 明確な意見の表明
  • 支持する理由や個人的な経験
  • セミフォーマルなレジスター（"Sehr geehrte"はなし、しかし超カジュアルでもない）
  • 元のフォーラムの質問への言及
  • 最低80〜100語

例のプロンプト：フォーラムのトピック："子供はスマートフォンを持つべきか？" 理由を添えて意見を表明する投稿を書いてください。

タスク3：正式メール

内容：機関、会社、または権威に宛てた正式な手紙またはメールです。特定の状況（苦情、問い合わせ、申請）に応じて応答し、3つのLeitpunkteに言及します。

試験官が期待すること：

  • 正式なレジスター："Sehr geehrte Damen und Herren"または"Sehr geehrte/r Herr/Frau [名前]"
  • 全体を通してSie形式
  • 3つのLeitpunkteがカバーされていること
  • 適切な正式なクロージング："Mit freundlichen Grüßen"
  • 最低80〜100語

B2ライティング：両方のタスクの説明

タスク1：フォーラム投稿（約150語）

内容：社会的または論争のあるトピックに関するオンラインディスカッションに参加します。明確な立場を示し、例や理由を用いてそれを支持する必要があります。

B1との主な違い：

  • より長いテキスト（150語対80〜100語）
  • 片方の立場を取る場合でも、賛成と反対の両方を含める必要があります
  • プロンプトに提供された他のフォーラムコメントに言及または反応することが期待されます
  • より複雑な議論：原因/結果、比較、仮定のシナリオ
  • より豊かな語彙と文の多様性が期待されます

例のプロンプト： "在宅勤務は法的権利であるべきか？" フォーラムの2つのコメントを読み、自分の意見を表明してください。例を挙げ、少なくとも1つの他のコメントに応答してください。

タスク2：正式メッセージ（約100語）

内容：職業的な文脈での正式なメール — 同僚、上司、または機関に宛てたものです。リクエストを行ったり、情報を提供したり、特定の状況に応じて応答することが多いです。

主な要件：

  • 適切な正式なレジスターと挨拶
  • 必要な情報を明確かつ簡潔に伝えること
  • プロンプトのすべてのLeitpunkteに言及すること
  • 適切な正式なクロージング
  • 約100語 — 書きすぎないこと

C1ライティング：両方のタスクの説明

タスク1：エッセイ（200語以上）

与えられたトピックについて構造化された議論的エッセイを書き、賛成と反対を論じ、結論に達します。トピックは通常、社会的なものです：教育における技術の役割、環境政策、ワークライフバランスなど。

C1エッセイでは以下が期待されます：

  • 明確な主張と構造化された議論
  • 多様な視点を持つ微妙な議論
  • 多様な文構造（複雑な従属、分詞構文、名詞化）
  • 学術的なレジスターと正確な語彙
  • 議論を統合した真の結論（単に質問を繰り返すのではない）

タスク2：正式な要約（80〜100語）

グラフ、チャート、または短いテキストからの情報を正式なレジスターで要約します。これは、データを簡潔かつ正確に伝える能力をテストします。

  • 客観的である必要があります — 個人的な意見は不要
  • 報告言語を使用すること："Aus der Grafik geht hervor, dass..."、"Die Daten zeigen..."
  • 80〜100語の制限を守ること — 長すぎたり短すぎたりすると得点を失います

試験官が実際に評価すること

評価基準を理解することは、ターゲットを絞った準備に不可欠です。ゲーテのライティング試験官は4つの主要なカテゴリを使用します：

1. Inhalt / Aufgabenerfüllung（タスクの達成度）

重み：高

  • すべての必要な内容ポイントに言及しましたか？
  • あなたの応答はプロンプトに関連していますか？
  • 十分なテキストを書きましたか？
  • 重要：1つのLeitpunktを欠くと、通常、この基準で1つのグレードバンドが下がります。書き終える前に、必要なポイントをすべてチェックしてください。

2. Kohärenz（コヒーレンスと構造）

重み：中-高

  • あなたのテキストは論理的に整理されていますか？
  • 文が自然に次の文に流れていますか？
  • 接続詞や移行語を使用していますか？
  • 明確な始まり、中間、終わりがありますか？

3. Wortschatz（語彙）

重み：中

  • あなたの語彙はレベルに適していますか？
  • 多様な語彙を使用していますか（同じ単語を繰り返さない）？
  • 単語は正確に使用されていますか？
  • 語彙の誤りは理解に影響を与えますか？

4. Strukturen（文法と構造）

重み：中

  • あなたの文は文法的に正しいですか？
  • 多様な文構造を使用していますか（単にSVOだけではない）？
  • B2+では、複雑な構造（従属節、受動態、Konjunktiv）を使用していますか？
  • 文法の誤りは理解に影響を与えますか？

重要な洞察：意味に影響を与えない軽微な文法の誤りは、テキストを不明瞭にする誤りよりもペナルティが少ないです。野心的な文法を使用し、いくつかの誤りがあるテキストは、単純で誤りのない文のテキストよりも多くのレベルで高得点を得ます。

得点が高くなる接続詞とフレーズ

適切な接続詞を使用することで、一貫性と語彙の幅を示します。以下はカテゴリ別の最も便利なものです：

意見を述べる

  • Meiner Meinung nach...（私の意見では...）
  • Ich bin der Ansicht, dass...（私は...という見解です）
  • Aus meiner Sicht...（私の視点からは...）
  • Ich bin davon überzeugt, dass...（私は...と確信しています）

議論を追加する

  • Außerdem / Darüber hinaus（さらに）
  • Hinzu kommt, dass...（加えて...）
  • Ein weiterer Aspekt ist...（もう一つの側面は...）
  • Nicht zu vergessen ist...（忘れてはならないのは...）

対比する

  • Einerseits... andererseits...（一方で...他方で...）
  • Trotzdem / Dennoch（それにもかかわらず）
  • Im Gegensatz dazu...（それに対して...）
  • Obwohl...（...にもかかわらず）
  • Zwar... aber...（確かに...しかし...）

原因と結果

  • Deshalb / Daher / Aus diesem Grund（したがって）
  • Das führt dazu, dass...（これは...につながります）
  • Die Folge ist...（その結果は...です）
  • Weil / Da（なぜなら / だから）

結論づける

  • Zusammenfassend lässt sich sagen...（要約すると...と言えます）
  • Abschließend möchte ich betonen...（最後に強調したいのは...）
  • Alles in allem...（全体として...）
  • Insgesamt...（全体的に...）

正式なメール用

  • Ich wende mich an Sie, weil...（私はあなたに連絡しています、なぜなら...）
  • Ich möchte Sie bitten...（あなたにお願いしたいのですが...）
  • Wären Sie so freundlich...（ご親切に...）
  • Für Rückfragen stehe ich gerne zur Verfügung.（質問があれば喜んでお答えします。）
  • Ich danke Ihnen im Voraus.（前もって感謝します。）

得点を失う一般的なミス

1. 不適切なレジスター

正式なメールで「du」を使用したり、非公式なメッセージで「Sie」を使用したりすること。これは、ドイツのコミュニケーションにおける社会的認識の欠如を示すため、最も罰せられるエラーの一つです。プロンプトに「Schreiben Sie an Ihren Freund」と書かれている場合は「du」を使用してください。「Schreiben Sie an die Firma」と書かれている場合は「Sie」を使用してください。

2. Leitpunkteの欠如

すべてのタスクには特定の内容ポイントがあります。時間が足りなかったり、忘れたりして1つをスキップすると、タスクの達成度スコアにとって壊滅的です。書く前に、必要なポイントをリストアップしてください。書いた後は、すべてがカバーされているか確認してください。

3. 書きすぎまたは書かなさすぎ

単語数が大幅に不足していると、十分な言語を生産できないことを示唆します。大幅に超えると時間を浪費し、より多くのエラーが含まれる可能性があります。最低限の10〜20％上を目指してください — これは範囲を示しつつ、過度に延長しないことを示します。

4. 構造や段落がない

段落の区切りがない単一のテキストブロックは、言語の質に関係なく、一貫性が低いと評価されます。明確な段落を使用してください：導入、本体（1〜2段落）、結論。

5. 同じ接続詞の繰り返し

唯一の接続詞として「und」や「aber」を使用することは、限られた言語範囲を示します。混ぜて使用してください：常に「aber」の代わりに「trotzdem」を使用し、常に「und」の代わりに「außerdem」を使用してください。

6. 接続詞の後の語順エラー

従属接続詞（weil、obwohl、dass、wenn）の後のドイツ語の語順は、動詞が最後に来る必要があります。「trotzdem」や「deshalb」のような副詞の後では、動詞が2番目に来ます（倒置）。これを間違えることは、B1〜B2で最も一般的な文法エラーの一つです。

7. 状況の文脈を無視する

プロンプトに「隣人が騒音について苦情を言った」と書かれている場合、騒音公害についての一般的なエッセイを書くべきではありません。記載された特定の状況に具体的に応答してください。試験官は、あなたのテキストが与えられた正確なシナリオに適切かどうかを確認します。

時間管理戦略

時間切れが最も多い理由で、候補者がライティングでパフォーマンスが低下します。以下は、分を割り当てる方法です：

B1：3タスクに60分

タスク時間活動の内訳
タスク1（非公式）15分2分計画、10分執筆、3分チェック
タスク2（フォーラム）20分3分計画、14分執筆、3分チェック
タスク3（正式）20分3分計画、14分執筆、3分チェック
バッファ5分最終レビューと修正

B2：2タスクに75分

タスク時間活動の内訳
タスク1（フォーラム、150語）45分5分計画、35分執筆、5分チェック
タスク2（正式、100語）25分3分計画、18分執筆、4分チェック
バッファ5分最終レビュー

C1：2タスクに80分

タスク時間活動の内訳
タスク1（エッセイ、200語以上）50分8分計画/アウトライン、35分執筆、7分チェック
タスク2（要約、80〜100語）25分5分ソース分析、15分執筆、5分チェック
バッファ5分最終レビュー

黄金のルール：計画段階を飛ばさないでください。2分のアウトライン作成は、10分の混乱した執筆と書き直しを節約します。

練習用サンプルトピックプロンプト

B1練習トピック

  • 非公式メール：あなたの友人が来週新しい仕事を始めます。お祝いのメッセージを書き、新しい職場について尋ね、一緒に祝うことを提案してください。
  • フォーラム投稿： "都市に住むのと田舎に住むのはどちらが良いですか？" あなたの意見と経験を共有してください。
  • 正式メール：オンラインで注文した商品が損傷して到着しました。会社に書いて、問題を説明し、交換をリクエストし、返品プロセスについて尋ねてください。

B2練習トピック

  • フォーラム投稿： "ソーシャルメディアプラットフォームは誤情報に対して責任を負うべきか？" 賛成と反対の議論をし、あなたの立場を述べてください。
  • 正式メッセージ：あなたは職場でチームイベントを企画しています。レストランにグループ予約、食事の選択肢、利用可能な日程について問い合わせるメールを書いてください。

C1練習トピック

  • エッセイ： "人工知能は教育にどの程度使用されるべきか？" 利点、リスクを論じ、バランスの取れたアプローチを提案してください。
  • 要約：2019年から2025年までのヨーロッパ諸国におけるリモートワークの割合の変化を示すグラフを要約してください。

タイマーを使ってこれらを練習し、可能であればチューターやネイティブスピーカーからフィードバックを受けてください。試験条件下でテキストを書くほど、プロセスが自然になります。しっかりとした学習計画には、5週目以降に週に2〜3回の完全なライティングタスクを含めるべきです。
